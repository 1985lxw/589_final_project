## Intro to ML Final Project

For this project, we conduct a comparative analysis of various machine learning algorithms across diverse datasets to evaluate performance, stability, and generalization.

### Algorithms Tested: 
- Neural Networks
- K-Nearest Neighbors
- Decision Trees
- Random Forests

### Datasets:
- Handwritten Digits: A high-dimensional dataset ($1,797$ samples, $64$ features) where pixel intensities act as numerical inputs for $10$ different labels.
- Oxford Parkinson's Disease: A purely numerical dataset used to observe how models handle high variance and potential overfitting in medical diagnostics.
- Rice Grain Images: A numerical dataset featuring morphological measurements where algorithms demonstrated high stability and consistent performance.
- Credit Approval: A mixed-feature dataset ($15$ features, $653$ samples) containing categorical attributes that require factorization for processing.
